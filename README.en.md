# bysj-051

#### Description
基于SSM的智慧医疗问诊系统+论文参考示例.系统角色：管理员、医生、患者
功能模块：管理员（患者管理、医生管理、科室管理、知识分类管理、预约挂号、病历信息等）、医生（预约挂号、病历信息、在线咨询）、患者（预约挂号、个人中心、在线咨询、评价、留言）
技术栈：SSM、Vue、Layui（门户web）等
测试环境：idea2024、navicat、MySQL5.7、Maven3
